# DinosaurList

> Centralized app store dashboard for discovering, accessing, and deploying WSD Penang internal software tools.

| Field    | Details                          |
|----------|----------------------------------|
| Author   | WSD Penang Engineering Team      |
| Platform | Windows Desktop                  |
| Storage  | Box Drive (Shared Link)          |

---

## Features

| Feature | Description |
|---------|-------------|
| **Centralized Tool Registry** | All WSD Penang internal tools registered in a single dashboard, synced with Box Drive shared storage. |
| **One-Click Deployment** | Launch or install tools directly from the dashboard — no manual file navigation or setup required. |
| **AI Assistant (Tool Recommender)** | NLP-powered chatbot that accepts natural language queries and recommends the most relevant tool using TF-IDF similarity matching. |
| **Tool Search & Filter** | Locate tools by name, category, keyword, or author using the built-in search interface. |
| **Tool Metadata Display** | Each listing shows name, version, author, supported formats, platform, and purpose at a glance. |
| **Box Drive Integration** | Tools are stored and shared via Box Drive — all engineers always access the latest version without manual syncing. |
| **New Tool Announcements** | Highlights recently added or updated tools so the team stays informed of the latest additions. |

---

## Use Cases

- **Forgot the tool name?** — Describe the task to the AI assistant and get an instant recommendation.
- **New team member onboarding** — Browse the full registry independently without asking colleagues.
- **Quick deployment on a new machine** — Locate any tool in DinosaurList and launch it in one click from Box Drive.
- **Version consistency** — The centralized registry ensures all entries always point to the current Box Drive path.
