TOOL            : DinosaurList
AUTHOR          : WSD Penang Engineering Team
PLATFORM        : Windows Desktop
FORMATS         : Box Drive (Shared Link)
PURPOSE         : Centralized app store dashboard for discovering, accessing, and deploying WSD Penang internal software tools
KEYWORDS        : app store, tool registry, dashboard, deployment, AI assistant, tool discovery, WSD Penang, Box Drive

Features

Centralized Tool Registry — All WSD Penang internal tools are registered and listed in a single dashboard, synced with Box Drive shared storage for easy access.
One-Click Deployment — Launch or install tools directly from the dashboard without manual file navigation or setup steps.
AI Assistant (Tool Recommender) — An NLP-powered chatbot that accepts natural language queries and recommends the most relevant tool based on your described need using TF-IDF similarity matching.
Tool Search & Filter — Quickly locate tools by name, category, keyword, or author using the built-in search interface.
Tool Metadata Display — Each tool listing includes its name, version, author, supported formats, platform, and purpose at a glance.
Box Drive Integration — Tools are stored and shared via Box Drive, ensuring all engineers access the latest version without manual syncing.
New Tool Announcements — Highlights recently added or updated tools so engineers stay informed of the latest additions to the ecosystem.

Use Cases

An engineer needs a data processing tool but cannot remember its name — they describe the task to the AI assistant and receive an instant recommendation.
A new team member wants to explore all available internal tools without asking colleagues — they open DinosaurList and browse the full registry.
An engineer needs to quickly deploy a visualization or QC tool on a new machine — they locate it in DinosaurList and launch it in one click directly from Box Drive.
The team lead wants to ensure everyone is using the latest version of a tool — the centralized registry ensures all entries point to the current Box Drive path.
